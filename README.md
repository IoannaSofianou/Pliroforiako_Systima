# YpoxreotikiErgasiaSept22_E19162_Sofianou_Ioanna
## Εγκατάσταση του Docker
1. Windows
   1. Από το link (https://hub.docker.com/editions/community/docker-ce-desktop-windows), κατεβάζουμε το πρόγραμμα 'Docker Desktop for Windows'
1. Linux
   1. Εκτελούμε στο terminal τις εξής εντολές:
     * `sudo apt-get update`
     * `sudo apt install -y apt-transport-https ca-certificates curl software-properties-common`
     * `curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -` 
     * `sudo add-apt-repository -y "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"`
     * `sudo apt-get update`
     * `sudo apt install docker-ce`

## Containerize Web Service
* Για την _δημιουργία_ του image, εκτελούμε την εντολή `docker build -t webservice` (το webservice είναι το όνομα του image)
* Για την _διαγραφή_ του image, εκτελούμε την εντολή `docker rmi webservice` (το webservice είναι το όνομα του image)
* Για την _εμφάνιση_ όλων των images, εκτελούμε την εντολή `docker images` 
* Για την _εμφάνιση_ όλων των ενεργών container, εκτελούμε την εντολή `docker ps -a`

## Εκτέλεση Εφαρμογής
### _Βήματα_
1. Κατεβάζουμε από το private repository το αρχείο: _YpoxreotikiErgasiaSept22_E19162_Sofianou_Ioanna-main.zip_
1. Κάνουμε το συγκεκριμένο αρχείο unzip
1. Ανοίγουμε το συγκεκριμένο αρχείο στο terminal: ( _**path**_: YpoxreotikiErgasiaSept22_E19162_Sofianou_Ioanna-main )
1. Για την εκτέλεση του container, πληκτρολογούμε την εντολή:
`(sudo) docker-compose up -d`
  1. Για τον τερματισμό όλων των υπηρεσιών και την διαγραφή του container, πληκτρολογούμε την εντολή: `(sudo) docker-compose down`      
* **Η εντολή _sudo_ χρειάζεται μόνο στην περίπτωση που η εφαρμογή τρέχει σε _Linux_**
1. Για την εκτέλεση του web service ανοίγουμε τον browser και πληκτρολογούμε το link:
(http://localhost:5000/HomePage)

## Απαιτήσεις Εφαρμογής
Το web service που υλοποιούμε έχει όνομα DS_Airlines και τρέχει, μέσω του Flask, στην πόρτα 5000, ενώ η MongoDB βρίσκεται στην πόρτα 27017.
Στην MongoDB δημιουργούμε μία βάση δεδομένων με όνομα DS_Airlines και 4 collections με όνομα:
1. Users
1. Flights
1. Booking
1. Admins

_Πιο αναλυτικά στο:_

* **1ο collection ('Users')** αποθηκεύονται οι απλοί χρήστες του συστήματος
* **2ο collection ('Flights')** αποθηκεύονται οι πτήσεις που δημιουργεί ένας διαχειριστής
* **3ο collection ('Booking')** αποθηκεύονται οι κρατήσεις εισιτηρίων που κάνει ένας απλός χρήστης
* **4o collection ('Admins')** αποθηκεύονται οι διαχειριστές του συστήματος
